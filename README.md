## Hello There

Pernahkah kalian mendengar istilah dart? Yap, saat ini saya belajar Dart untuk pembuatan aplikasi menggunakan framework Flutter. Silahkan pelajari lebih lanjut [Dart](https://dart.dev/)

## Installation

1. Clone project ini ``git clone https://github.com/fauzan2720/Belajar-Dart.git``
2. Buka file .dart
3. Pastikan anda menginstall extension Dart pada VSCode
4. Running file Dart dengan ketik **F5**
5. Happy Coding :)
